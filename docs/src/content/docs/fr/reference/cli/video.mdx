Certaines des [fonctionnalités de traitement vidéo](/genaiscript/reference/scripts/videos) sont également disponibles dans la CLI.

### `video probe`

Affiche le résultat de `ffprobe` dans la console.

```sh
genaiscript video probe myvid.mp4
```

### `video extract-audio`

Extrait l’audio dans un format plus léger, optimisé pour la transcription.

```sh
genaiscript video extract-audio myvid.mp4
```

### `video extract-frames`

Extrait des captures d’écran de la vidéo. Vous pouvez spécifier les timestamps en secondes ou sous le format `h:mm:ss`, ou encore un nombre de captures.

```sh
genaiscript video extract-video myvid.mp4
```
