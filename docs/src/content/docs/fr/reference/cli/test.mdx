Exécute les tests dans les scripts en utilisant [promptfoo](https://www.promptfoo.dev/).

```bash
genaiscript test "<scripts...>"
```

Vous pouvez remplacer les modèles utilisés pour les tests à l’aide de `--models` :

```bash "--models openai:gpt-4 ollama:phi3"
genaiscript test "<scripts...>" --models openai:gpt-4 ollama:phi3
```

:::note
Cette fonctionnalité nécessite d’ajouter `@genaiscript/api` à vos dépendances dans `package.json`.
:::

## visionneuse de résultats

Exécutez la commande `test view` pour lancer la visionneuse des résultats de test :

```bash
npx genaiscript test view
```
